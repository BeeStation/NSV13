  • Renamed Hardware Configuration to Settings.
  • Improved tablet flashlight UI interaction with cyborg code.
  • Added logging to PDA bombs.
  • Brig physician now starts with a cartridge.
  • Added a locator app for janitors to find mops, cleanbots, and janicarts with.
  • Improved ModPC behavior with sechuds showing the wrong HUD.
  • You can now use holochips directly on a PDA/tablet to insert credits to the ID inside.
  • Tablet flashlight now uses movable lighting instead of jumping around
  • Tablets can be used while inside lockers.
  • Changed tablet program access requirements to support lists.
  • Added an emoji entry TGUI interface. For use with the mime PDA.
  • Emagging a modPC now opens a console window with flavor text and sets the theme to syndicate.
  • Fixes PDA sent photos opening an undersized window.
  • Cyborg integrated tablet now has a button background.
  • AI now has a small integrated tablet only for sending messages.
  • Added an improved photo selection TGUI menu, mainly for use by silicons.
  • Photos now describe their dimensions properly.
  • Photo descriptions no longer randomly place periods around.
  • Photos can no longer describe invisible things.
  • Nukie tablets now start with the Remote Airlock Control program, for remotely interfacing with the infiltrator's blast door.
  • Regular cargo techs no longer have the Ore Silo logs in their PDA Cartridge. It's now locked to the QM.
  • Added a new Cargo Technician cartridge with its own icon.
  • Added a Log Viewer program for viewing ore silo and NTChat logs on the filesystem.
  • Modular computers now correctly show visible_messages to the user if they are holding it.
  • Removed the messaging console's system message capability.
  • The Status Display modPC app can now update status alert images.
  • Fixed some runtimes in Cyborg Self Monitoring and Robot Control.
  • D.E.T.O.M.A.T.I.X. charges are no longer infinite.
  • D.E.T.O.M.A.T.I.X. explosions now do the same damage as before tablet-PDAs.
  • Power-ON disks are now improper nouns.
  • changed the fingerprint to DNA in the medical records app in PDAs
  • added scrollbars to the direct messenger PDA app
  • Converted the message monitor to TGUI
  • The Message Monitor can now send admin messages with emojis and images.
  • Changed the flavor text for the monitor decryption key paper.
  • Emagging the message monitor no longer immediately prints the key, but instead prints the key when the message monitor is usable again (after 10 * keylength seconds). The key couldn't be used prior to this previously, so it just makes it more consistent.
  • Fixed PDA messages with images not always properly line breaking inside "View Messages" on the Messenger app.
  • Fixed a possible hard-del when the message monitor's linked server was deleted.
  • Added logging and admin messages for deletion of PDA/request logs.
  • Improved the Message Monitor's server linking behavior to support multiple servers of the same network and name.
  • Fixed an exploit where you could unlink the active message monitor server, link to another with a different decryption key, and still be authenticated.
  • Added an admin verb, "Send PDA Message"
  • relocated cartridge.dm to job_disk.dm. repathed cartridge vendor to /job_disk
  • HoP disk vendor sells various job disks now. there're contraband inside.
  • HoP disk vendor no longer sells PDAs that have weirdly-pre-inserted-head-only-job-disks. It sells normal PDAs now.
  • Changed the default installed PDA programs to only being installed on the PDA subtype instead of the default Tablet SSD.
  • Changes the size of the Notepad to 0 and makes it undeletable and unavailable on NTNet.
  • Moves the Wiki Uplink Program to only being installed by default on the PDAs
  • Added name and job searching to the PDA Messenger contacts panel.
  • PDA message inputs are now TGUI-based.
  • When inputting a new PDA ringtone, the old one will be in the input text box already.
  • Improved some PDA messenger code.
  • TGUI text inputs now properly autofocus text.
