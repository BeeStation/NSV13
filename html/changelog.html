21 February 2026

DeltaFire15 updated:

  • Small craft can now be docked by valid craft initiating collision with them, as opposed to it having to be the other way around, provided docking mode is enabled. Doing it this way around forcibly enables the small craft's brakes if it lacks a working pilot.
  • Escape pods no longer have an access requirement (to eject occupants via ID).
  • Fighter escape pods being destroyed no longer messes up the occupant(s)'s movement once revived.
  • Fighter escape pods no longer drop part of their equipment on creation (and as such actually have a canopy).
  • Weapon selection dropping no longer drops a single runtime if called by a Fighter being destroyed.
  • Fighter escape pods being created on a non-overmap z no longer runtimes.
  • Overlay for escape pods missing their canopy.
  • You can no longer summon the escape shuttle via a certain exploit.
  • AI traitors now have an individual min pop config entry (AI_TRAITOR_MIN_PLAYERS)

10 January 2026

DeltaFire15 updated:

  • Multiple "entered combat" system announcements at once will no longer stack their noise.
  • The highjack description isn't out of place anymore.
  • Highjack bleeding is no longer fake.
  • Shower frames now inform you that you can turn them into a blast shower.

09 January 2026

DeltaFire15 updated:

  • Traders in the badlands no longer fail to generate their stock.
  • Space Yellow pages of neutral stations no longer runtime when sold.
  • Space yellow page info of stations now properly cleans up when a new iteration is generated.

03 January 2026

DeltaFire15 updated:

  • Equipping the Naked outfit now equips the real Naked outfit.
  • The muni pointer should now be more resistant to null entries.
  • The plasma loader machine now properly drops its board.
  • The plasma loader board is now protected.
  • The plasma caster board is now better protected.
  • The plasma caster & loader boards can now be tracked.
  • Singularities no longer infinitely suck out material from reinforced floors.
  • Reinforced floors are now more resistant to ranged singularity pulls
  • Openspace is now unaffected by singularities.
  • Relayed projectiles now have a higher degree of targeting randomness.
  • Relayed projectile starting sides are now determined by the angle of the original projectile.
  • Projectile impact quadrants are now determined by projectile facing rather than relative position to ship (hopefully feels more accurate).

Pockets-byte updated:

  • changes the aetherwhisp bartender shotgun to having a standard firing pin
