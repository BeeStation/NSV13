01 December 2025

DeltaFire15 updated:

  • READ THIS: New / adjusted keys: TAC: "F" can filter missile weapon ammo when active; Pilot can also switch between active weapons; 1-4 keys can select weapons for either operator.
  • Refactored everything about ship weapon datums. Elaborating would fill this changelog, check the PR.
  • Weapon order sorting can now be adjusted via TAC consoles.
  • Ghost ships now use the real TAC UI.
  • The aiming laser should now be more centered.
  • Aiming beam trajectory should behave the same as the actual projectiles fired.
  • ghost ship pilots can restart piloting if they somehow clicked the end piloting button.
  • AI ship maximum ammo is no longer implied by their initial ammo, instead being a seperate var.
  • Fighter air-to-air repair works on big ships now like it says it does (albeit much slower).
  • Broadside guns no longer runtime if you crowbar them.
  • The broadside gun machinery firing call is no longer malformed.
  • Probe torp projectiles once again have an icon.
  • Overmap pixel projectile firing should now be less janky.
  • Hybrid Railguns no longer runtime on fire due to a malform firing argument.
  • Unloading already empty weapons no longer runtimes.
  • The plasma caster no longer runtimes if created without an adjacent loader / its loader being deleted is now accounted for (can be linked via multitool).
  • Space mines no longer runtime when affecting an area-less ship.
  • The TAC console no longer yanks you back into control under some circumstances.
  • Some interior-exterior Fighter access attempts weren't caught before.
  • A lot of overmap related angle stuff should be less jank now.
  • Ghost ships are now semi-properly added to their systems.
  • Holomaps don't runtime on component destroy sometimes.
  • Prison grilles no longer runtime when exploded.
  • Throwmode doesn't runtime when keybind is used on weird hudded mobs (e.g. ghost ship pilots).
  • The MPAC projectile now behaves somewhat saner.
  • The overmap clickdrag procs no longer brick horribly if you mouse over chat.
  • The overmap gunning component no longer does expensive clickdrag reactions if not automatic (reads: for nothing).
  • Fighters should no longer runtime during destruction if their canopy was not broken beforehand
  • You cannot control space rocks.
  • The MPAC projectile no longer multihits targets on each tile move (it can still multihit them on seperate tile moves as is intended).
  • Turf piercing projectiles aren't always destroyed when intercepting missiles.
  • The MPAC projectile is much more resilient to guided warheads.
  • The MPAC projectile no longer relays to overmaps.
